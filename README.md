# multi-llm-bus

Общая шина и LLM Wiki для трёх моделей: Claude (оркестратор), Gemini и Grok.

## Цель
- Высокопропускная связь между моделями без постоянного copy-paste.
- Живая коллаборативная Wiki и база знаний.
- Версионирование всех артефактов.

## Структура
- `/inbox` — входящие сообщения и задачи
- `/outbox` — исходящие
- `/state` — текущее состояние, Current State, протокол
- `/reports` — финальные отчёты задач
- `/protocol` — версии Multi-LLM Protocol
- `/wiki` — структурированная LLM Wiki (знания, лучшие практики, примеры)

## Как использовать
- Claude и Gemini редактируют через Drive (синхронизируется).
- Grok читает/пишет через коннекторы.
